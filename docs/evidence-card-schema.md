# Evidence Card Schema

Evidence cards are the primary export format for analyst findings. They are generated by:

```bash
uv run radiancefleet export evidence --alert 42 --format md
uv run radiancefleet export evidence --alert 42 --format json
```

Or via API: `POST /api/v1/alerts/{id}/export?format=md`

## Gate: analyst review required

Export is **blocked** when `status == "new"`. This enforces the analyst review gate (NFR7 — no raw-machine-output publication). Set status to `under_review`, `documented`, or `needs_satellite_check` first.

---

## Field reference

| Field | Type | Description |
|-------|------|-------------|
| `alert_id` | int | Gap event ID |
| `vessel.mmsi` | string | 9-digit MMSI |
| `vessel.imo` | string\|null | IMO number |
| `vessel.flag` | string\|null | 2-letter ISO flag state |
| `gap.start_utc` | ISO datetime | AIS gap start |
| `gap.end_utc` | ISO datetime | AIS gap end |
| `gap.duration_minutes` | int | Gap length in minutes |
| `risk.score` | int | 0–100 composite risk score |
| `risk.breakdown` | object | Per-signal contributions |
| `movement_envelope.max_plausible_distance_nm` | float\|null | Maximum physically possible distance at max speed |
| `movement_envelope.actual_gap_distance_nm` | float\|null | Distance between last and first AIS points |
| `movement_envelope.velocity_plausibility_ratio` | float\|null | `actual / max_plausible` — >1.0 means impossible |
| `movement_envelope.impossible_speed_flag` | bool | True if ratio >1.1 |
| `last_known_position` | object\|null | Last AIS fix before gap |
| `first_position_after_gap` | object\|null | First AIS fix after gap |
| `satellite_check_status` | string | `not_checked` / `candidate_scenes_found` / `reviewed` |
| `satellite_scene_refs` | array | Sentinel-1 scene IDs |
| `analyst_notes` | string\|null | Analyst free-text notes |
| `status` | string | Alert status at export time |
| `corridor_name` | string\|null | Corridor the gap is in |
| `coverage` | [quality, description] | Regional AIS coverage quality |
| `exported_at` | ISO datetime | Export timestamp |
| `disclaimer` | string | Mandatory disclaimer text |

## Markdown rendering

The Markdown format renders the same fields in human-readable sections:

1. **Vessel** — MMSI, IMO, name, flag, type
2. **AIS Gap** — start, end, duration
3. **Risk Score** — total + per-signal breakdown
4. **Movement Envelope** — max plausible distance, actual distance, velocity ratio
5. **AIS Boundary Points** — last known position + first position after gap
6. **Satellite Check** — status + scene references
7. **Data Source Coverage** — regional AIS quality with caveat
8. **Analyst Notes** — free-text
9. **Disclaimer** — mandatory text

## Mandatory disclaimer

All evidence cards must include:

> DISCLAIMER: This is investigative triage, not a legal determination. This tool identifies patterns warranting further investigation. No conclusions about sanctions violations or criminal activity should be drawn from this output without independent expert verification.
